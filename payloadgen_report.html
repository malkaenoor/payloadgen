5. Implemented Module: Encoder

✔ Completed (Base64, URL, HTML encoding)

encoder.py

import base64, urllib.parse, html class Encoder: def encode(self, payload, method): if method == "base64": return base64.b64encode(payload.encode()).decode() if method == "url": return urllib.parse.quote(payload) if method == "html": return html.escape(payload) return None
